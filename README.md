# Predicting Tesla's Next-Day Stock Return: A Linear Regression Study of Daily Price Data

## Overview

This project explores the use of Linear Regression to predict Tesla's (TSLA) next-day stock return using historical daily price data. The objective is to investigate whether past price movements can provide useful information for forecasting future returns and to evaluate the performance of a simple machine learning model in financial prediction.

## Objectives

- Collect and analyze historical Tesla stock price data.
- Calculate daily stock returns.
- Engineer features from historical price movements.
- Train a Linear Regression model to predict next-day returns.
- Evaluate model performance using regression metrics.

## Dataset

The dataset consists of Tesla (TSLA) daily stock price data, including:

- Open Price
- High Price
- Low Price
- Close Price
- Adjusted Close Price (if available)
- Trading Volume

Daily returns are calculated from closing prices and used as the target variable for prediction.

## Methodology

### Data Preprocessing

- Handle missing values.
- Calculate daily returns.
- Create lagged return features.
- Split data into training and testing sets.

### Model Development

- Implement a Linear Regression model using Python.
- Train the model on historical Tesla stock data.
- Generate predictions for next-day stock returns.

### Model Evaluation

Performance is evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Results

The project assesses the ability of a Linear Regression model to capture patterns in Tesla's historical stock data and highlights the challenges associated with predicting stock market returns.
